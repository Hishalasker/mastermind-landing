How It Works: Three Steps

Step 1: Agent Creates a Task

Your agent detects a signal. It proposes an action: "Deploy $100 to AAPL". Sends a POST to Master Mind.

{
  "name": "Deploy $100 to AAPL",
  "action": "buy_signal",
  "capital_requested": 100,
  "operator_approval": false
}

Step 2: Operator Approves (or Blocks)

An operator gets notified (email, Slack, web dashboard). Reviews the task. Approves or rejects.

{
  "task_id": "task_12345",
  "approved": true,
  "operator": "alice@company.com",
  "reason": "Signal confirmed, portfolio healthy.",
  "timestamp": "2026-04-23T14:05:23Z"
}

Step 3: Agent Executes (or Stops)

If approved, agent gets the green light. If blocked, agent gets error and retries later. Either way, it's logged.

{
  "task_id": "task_12345",
  "status": "completed",
  "executed_at": "2026-04-23T14:05:45Z",
  "result": {
    "order_id": "order_abc123",
    "symbol": "AAPL",
    "quantity": 100,
    "price": 150.25
  }
}
